數值積分程式 (Python)
使用梯形法與辛普森法計算定積分，並繪製積分曲線圖示。

程式功能說明
計算定積分
使用 梯形法 (Trapezoidal Rule) 計算積分近似值
使用 辛普森法 (Simpson Rule) 計算積分近似值
可比較兩種數值方法的精度與誤差
支援任意函數
預設函數 f(x) = sin(x)
使用者可自由修改 f(x) 為其他連續函數，如 x**2, cos(x), exp(x) 等
積分誤差分析
顯示數值積分結果與精確值之差
方便觀察不同方法的精確度
圖形化
使用 matplotlib 繪製函數曲線
填充曲線下方面積，直觀表示積分面積

程式使用方式
開啟程式 numerical_integration.py
選擇函數 f(x) 與積分上下限 a, b
設定分割區間數 n (數值越大，精度越高)
執行程式：
python numerical_integration.py

程式將輸出：
梯形法積分結果
辛普森法積分結果
精確值（若已知）
各方法誤差
整體積分曲線圖示
